# Passw_Manage
///CryptoSafe Manager — кроссплатформенный менеджер паролей с графическим интерфейсом, зашифрованной локальной базой данных и защищённой обработкой буфера обмена///

Technology Stack:
* Язык програмирования: Python (most accessible, great crypto libraries)
* Криптографические библиотеки: cryptography, pycryptodome, argon2-cffi
* Защита буфера обмена: Custom secure clipboard with auto-clear 

SPRINT 1.
Основа – Защищенная база данных и оболочка с графическим интерфейсом
Цель: Создать зашифрованную базу данных SQLite и базовое окно с графическим интерфейсом.
Результаты:
Схема SQLite для хранилища (записи, журнал аудита, настройки).
Шифрование конфиденциальных полей на прикладном уровне перед вставкой в базу данных.
Главное окно с меню, списком и кнопками-заполнителями.
Ключевая особенность: Первый запуск программы установки с созданием мастер-пароля.
